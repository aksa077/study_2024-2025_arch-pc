# Лабораторная работа №3
## Тема: Язык разметки Markdown

### Цель работы
Изучение языка разметки Markdown и его использования для создания отчётов.

### Описание работы
- Создан файл отчёта в формате Markdown.
- Преобразован отчёт в PDF и DOCX с помощью Pandoc.

### Выводы
Markdown — это удобный инструмент для написания текстов с возможностью конвертации в разные форматы.
