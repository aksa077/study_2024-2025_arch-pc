# Лабораторная работа: Работа с ассемблером и Git

## Цель работы
Изучить основы работы с ассемблером, научиться разрабатывать, собирать и запускать программы на ассемблере для архитектур x86_64 и aarch64. Также освоить базовые команды для работы с Git и GitHub.

## Выполнение работы
- Использовалась Ubuntu на архитектуре ARM (aarch64).
- Написана программа на ассемблере, которая выводит строку "Hello, World!".
- Код программы адаптирован под архитектуру aarch64.
- Выполнены трансляция и компоновка программы.
- Локальный репозиторий настроен и подключён к удалённому, файлы загружены на GitHub.

## Заключение
- Работа помогла изучить особенности ассемблера.
- Улучшены навыки работы с Git и GitHub.
